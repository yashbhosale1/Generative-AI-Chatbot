Generative AI Chatbot

Built with OpenAI API, Flask, LangChain, HuggingFace (optional), and scikit-learn intent detection.

  • Open Chat Interface
  • Files: app.py, utils.py, intents.json

How it works (summary)

  1. User input -> NLP preprocessing (cleaning & tokenization)
  2. Intent detection (scikit-learn logistic regression)
  3. If confident: return canned response
  4. Else: call OpenAI (via LangChain if installed) to generate an answer

To run: see README or open terminal and run python app.py after installing requirements and setting OPENAI_API_KEY.
